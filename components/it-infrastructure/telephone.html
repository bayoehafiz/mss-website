1. Analog Telephone System

With Analog Phone, you can get good voice quality because it uses a cable, do not require much power, even if the network goes down, the telephone network is still on, flexibility PBX support.

Why Analog System?

Analog systems have supported businesses for decades. Built on standard copper wire phones, they are reliable, boast good voice quality, and have the basic features you might find in a typical home phone such as hold, mute, redial, and speed dial. They may also be able to transfer calls between extensions.

2. IP-Based Telephone System

Telephone System Solution IP based is the network concept communication next generation network that can integrate conventional telephone network, mobility telephone network, satelite telephone network and network based on package.
For this solution, we provide 2 type of telephone from different vendor, there are : MyPBX YeastarU100 for IP PBX system and Yealink SIP-T21PE for IP Phone system.

The benefits of using IP Phone

IP telephone system is easier to install, configure, and maintain, flexible to move other point, scales up or down easily, employees' numbers follow them wherever they go, a range of call features are supported, hosted IP saves businesses money, integrates with other business systems


3. Hybrid Telephone System

Hybrid system work in the PBX system that can be integrated with network IP Data, and this is one of the solution that reduce cost, because telephone communication can be connected with VoIP, WAN or LAN that located in internal company with no worries about communication quality.
We use Panasonic NS300 and Panasonic DT-543 and Panasonic KX-TS505X for our hybrid system.

Why you need hybrid system

A hybrid PBX has many of the same great features you can expect from VoIP services, including call rules, auto-attendant, and hold queues. The hybrid PBX also has the capacity for virtual conference rooms, fax over IP, and video conferences. The hybrid PBX also can do SIP trunking.
